WHERE create_continuation(card_id, ...) FITS
Same three touch points in mcp-gateway (tools.mjs implementation + discoverTools declaration gated on kanbanAvailable() + dispatchTool line). Implementation is POST `${kanbanBase}/cards` (handleCreateCard, kanban-loop/scripts/server.mjs:856-905; route :1948). Accepted payload today: title, description, project, goalMode, acceptance, workKind, phases, tier/classification.tier, origin, originChannel {channel, threadId}, duty, level, sequence (the D15 door-1 resolved-sequence persistence), outpost, quick.
